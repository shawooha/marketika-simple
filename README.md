# Marketika Landing Page HTML Template

Marketika is a freebie landing page template you can download elsewhere.

I've made it for training purposes.

No grid, no flex, good old float/clearfix :D

Tested on Firefox and Chrome.

NOTE: this version is not adaptive.

Compiled on VS Code with "Live Sass Compiler" and "Live Server" extensions.

Live Sass Compiler options:

```"liveSassCompile.settings.generateMap": false,
    "liveSassCompile.settings.formats":[
        // This is Default.
        {
            "format": "expanded",
            "extensionName": ".css",
            "savePath": "/build/css"
        },
        // You can add more

            "extensionName": ".min.css",
